# 图形化控制面板使用指南

## 🎛️ 控制面板功能

现在粒子系统包含一个完整的图形化控制面板，可以实时调整所有重要参数！

### 可调参数

1. **重力 (Gravity)**

   - 范围：0 - 50
   - 默认值：10
   - 说明：控制粒子向下的重力强度
   - 提示：设置为 0 可以创建零重力环境

2. **阻尼 (Damping)**

   - 范围：0.95 - 1.0
   - 默认值：0.995
   - 说明：速度衰减系数，越接近 1 粒子运动越持久
   - 提示：
     - 0.95 = 快速衰减，粒子很快停止
     - 1.0 = 无衰减，粒子永动（不推荐）

3. **粒子数量 (Particle Count)**

   - 范围：100 - 2000
   - 默认值：800
   - 说明：场景中粒子的总数量
   - 提示：
     - 数量越多，碰撞检测计算量越大
     - 建议根据设备性能调整
     - 修改后需要点击"重置粒子"按钮

4. **粒子半径 (Radius)**
   - 范围：2 - 15 像素
   - 默认值：8
   - 说明：粒子的最大半径
   - 提示：半径越大，碰撞越明显

### 🎮 交互方式

- **调整参数**：拖动滑块实时改变值
- **重置粒子**：点击"🔄 重置粒子"按钮应用新的粒子数量和半径设置
- **隐藏面板**：按键盘 `H` 键切换控制面板显示/隐藏
- **鼠标交互**：点击并拖动鼠标推动粒子

### 📊 实时参数

以下参数可以实时生效，无需重置：

- ✅ 重力
- ✅ 阻尼

以下参数需要点击"重置粒子"才能生效：

- ❌ 粒子数量
- ❌ 粒子半径

控制面板采用现代化设计：

- 半透明黑色背景
- 高斯模糊效果（backdrop-filter）
- 渐变色滑块
- 平滑的动画过渡
- 响应式提示信息

### 💡 使用技巧

1. **性能优化**

   ```
   粒子数量: 500-800 (流畅)
   重力: 5-15 (自然效果)
   阻尼: 0.99-0.995 (真实物理)
   ```

2. **创意效果**

   - **爆炸效果**：高重力 + 低阻尼
   - **漂浮效果**：零重力 + 高阻尼
   - **弹珠效果**：中等重力 + 极高阻尼(0.999)

3. **压力测试**
   ```
   粒子数量: 1500-2000
   观察 FPS 变化
   ```
